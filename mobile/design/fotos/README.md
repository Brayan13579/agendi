# Fotos reales

Guarda aquí fotos reales para usar en la app: barbería (local, sillas, productos), barberos, clientes (con permiso), cortes/trabajos terminados, etc.

Si tienes muchas, puedes organizarlas en subcarpetas, por ejemplo:
- `local/`
- `barberos/`
- `trabajos/`
- `clientes/`

Cuantas más tengas, mejor se puede armar un diseño que se sienta "real" y no genérico.
