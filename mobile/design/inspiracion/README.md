# Inspiración

Guarda aquí capturas, imágenes o links de diseños que te gusten (estilos, layouts, animaciones, paletas, etc.).

No importa si no son del mismo tipo de app — lo que sirve es la sensación visual: tipografía, composición, cómo se mueven las cosas al hacer scroll, mezcla de imágenes y texto, etc.

Cuando agregues algo, cuéntame qué es lo que te llamó la atención de esa imagen en particular (¿la tipografía? ¿cómo se sobreponen las fotos? ¿el color? ¿la animación?). Eso me ayuda a traducirlo a la app sin adivinar.
